선언위치

🍊 데이터 종류

  • String: 문자형 ("" 따옴표)
  • Number: 숫자형
  • Boolean: 참,거짓
  • Undefined: 값이 할당되지 않은 상태
  • Null: 의도적으로 비어 있음을 의미
  • Array: 여러 데이터를 순차적으로 저장 (같은 데이터값끼리 지향)
    • 접근: index로 접근한다. ex) kdt[3]
    • 지향: 데이터 타입이 동일한 것끼리 배열을 만들자
    • push, pop, unshift
    • split: 문자를 인수를 기준으로 쪼개서 배열로 반환
      ex) let helloSplit = hello.split('');
    • reverse: 배열의 순서를 뒤집어서 반환
    • join: 배열을 인수 기준으로 병합해서 문자열을 반환

  • Object: 여러 데이터를 key: value 의 형태로 저장
    • 여러 데이터를 담을 수 있다.
    • 접근: .name / ['age']
    • 추가: superMan['job'] = '기자';
    • 삭제: delete superMan.weight;
    • 존재 여부 확인: console.log('name' in pororo);
    • obj를 위한 for문: for (key in pororo) {}
    • method: 객체 프로퍼티로 할당된 함수
    • this
    • 화살표 함수와 this의 관계
      화살표 함수는 this를 가질 수 없다. ( 더 상위 객체를 호출 )

  • typeof: 데이터의 타입을 알려줘
    - null/obj 모두 object로 출력되는 자바스크립트의 단점!!!
